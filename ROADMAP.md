# Roadmap

The Nadagotchi project is an evolving digital life simulation. This roadmap outlines the high-level goals and planned features for future updates.

## Current Focus: UX Polish & System Robustness
- [x] **Config Centralization**: Move magic numbers to `Config.js` (Completed v0.6.0).
- [x] **UX Improvements**: Add better feedback for disabled actions (Completed).
- [x] **Quest System**: Implement data-driven Quest System (Completed).
- [ ] **Accessibility**: Add keyboard navigation support for all modals.
- [ ] **Responsive Design**: Improve layout scaling for mobile devices.

## Upcoming Milestones

### Milestone 1: Enhanced Social & World
- **NPC Schedules**: NPCs move between locations based on time of day.
- **Dynamic Quests**: Procedurally generated quests from NPCs.
 - [x] **Friendship Decay**: Relationships degrade over time if ignored.

### Milestone 2: Advanced Career Paths
- **New Careers**: Chef, Musician, Gardener.
- **Career Promotion**: Levels within careers (Apprentice -> Master).
- **Workplace Events**: Random scenarios during work minigames.

### Milestone 3: The "Cloud" Era (Meta-Game)
- **Pet Exchange**: Export/Import pets via QR codes or text strings.
- **Leaderboards**: Hall of Fame integration with online high scores (optional).
- **Ghost Pets**: Visit the "graveyards" of other players' retired pets.

## Long-Term Vision
- **Visual Overhaul**: Pixel art animations for all actions.
- **Sound Design**: Dynamic soundtrack that changes with mood and season.
- **Mobile App**: PWA wrapper for native mobile experience.
